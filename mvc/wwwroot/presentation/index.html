Security & Authentication

Enterprise-Grade Security Features

🔐

JWT Authentication

Token-based authentication with 2-hour expiration. Secure API access for all endpoints.

👥

Role-Based Access (RBAC)

Three-tier permission system. Each role has specific access rights and restrictions.

🍪

Secure Cookies

HttpOnly, Secure, and SameSite=Strict cookies. Protection against XSS and CSRF attacks.

🔒

Password Security

Hashed password storage. Secure password reset flow via email.

🌐

Domain Restrictions

Domain-based access control. Custom restrictions for specific organizations.

⏱

Session Management

Automatic token expiry handling. Secure logout with cookie cleanup.
